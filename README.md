# 안녕하세요. 게임 개발자 하건입니다.

Unity와 C#을 활용해 게임 시스템과 구조 설계를 중심으로 개발하고 있습니다.  
확장 가능한 코드 구조와 재사용 가능한 라이브러리 제작에 관심이 있습니다.

---

## 기술 스택

- C#
- Unity
- Git
- Unity Editor Tool 제작
- 게임 시스템 설계

---

### Unity 공용 프레임워크

기존 프로젝트에서 공용으로 사용할 수 있는 기능들을 리팩토링하여 만든 프로젝트입니다.

- UI, 사운드 및 각종 유틸 라이브러리
- 유닛 (플레이어 및 몬스터 기본 클래스)
- 버프 및 스킬 시스템
- 공격 오브젝트 및 투사체
- 저장 및 테이블 데이터 Json 연동
- 복잡한 AI용 행동트리 툴
- 아이템 및 인벤토리
