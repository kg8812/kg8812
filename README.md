## 주요 프로젝트

### 가제 : WinterSpring

2년동안 대규모 팀으로 진행한 2D 메트로베니아 게임프로젝트입니다.
프로젝트에서 개발팀장을 맡아 다양한 게임 시스템 설계와 개발을 담당했습니다.

주요 구현

- 섹터 동적로딩 시스템
- 플레이어 스킬 및 스킬트리
- 무기, 악세서리 시스템
- 비개발자용 섹터 제작 툴 개발
- 스파인 애니메이션 및 이펙트 적용 시스템
- 게임 데이터 관리 시스템

등 다양한 시스템을 구현했습니다.

Repository  
[링크](https://github.com/kg8812/WinterSpring)
